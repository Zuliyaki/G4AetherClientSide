Diagnosis Management: Help

General description

Login as a patient, you can make search from with different types of filters, he can click on any of the result table rows so he can read information about your mental disease and lool at your diagnosis treatments, there is also a print button in order to print it, login as a psychologist you can make all of mentioned above and all the Create/Modify/Delete funtions

Behaviour

Create a Diagnosis

To create a diagnosis, you need to be loged in as a psychologist, you need to right click on the Diagnosis table and selected the Create new Diagnosis on the menu that shows. the diagnosis will be created and you will need to modify it with the needed values on each cell, the patient cell will show a box with all the patients DNI, the psychologist cell will be created with your DNI, the Date of creation cell will show the date of today, the on therapy checkbox will be set false, and the last treatment change date cell will set at today date

Modify a Diagnosis

In oder to modify you need to click on the diagnosis you want to modfiy and each cell will be editable, you can change the patient, and select a new patient, the date of creation cannot be edited, the psychologist cell can not be edited it will be yours, the cell on therapy you can check it or uncheck it, and you can change the last treatment change date cell and set a new one with the datepicker

Delete a Diagnosis if you want to delete a diagnosis you need to select the row of the diagnosis you want to delete and right click on it, a Menu will show and you need to select the

Delete Diagnosis option, it will be removed and the table will be updated.

Search Diagnosis To search a Diagnosis you need to frist select a search method on the right top combobox that is labeled as search method, ifFind all diagnosis all fields will be deactivated, there is nothing to worry about. When selecting Find all diagnosis by patient id, the PatientDNI field must be filled. and if Find all diagnosis by patient on therapy also the PatientDNI field must be filled. when selecting Find all notes beetween dates and patient id, the From, to, and PatientDNI field will enable and it will need to be filled, the date on to cannot be before the data on From, In order to activate the search button the requested fields will have to be filled it will need to be filled in order to activate. Once the actions mentioned above are finished, click on the Search button. If there is no error the diagnosises will apear on the table shown in the window. Print Diagnosis current info

Pressing the Print button will pop up a document showing the diagnosis on the table.

Exit the application

On the top Menu, selecting User menu, two options will be shown, selecting the Log out you will be asked for confirmation.

Go to another Window

Going into the top Menu and selecting any of the menus, it will show a menu to go to that page, selecting it, the new window will pop up
